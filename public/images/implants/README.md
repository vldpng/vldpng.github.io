# Изображения имплантов для калькулятора

Положите сюда 3 файла в формате **WebP** с прозрачным фоном (вертикальный имплант по центру):

| Файл                | Бренд      |
|---------------------|------------|
| `megagen.webp`      | Megagen    |
| `root.webp`         | ROOT®      |
| `straumann.webp`    | Straumann  |

Рекомендации:
- прозрачный фон (alpha), вертикальная ориентация;
- примерно 400×800 px (соотношение ~1:2), вес — десятки КБ;
- одинаковый ракурс и кадрирование у всех трёх, чтобы смотрелись единообразно.

Пока файлов нет, калькулятор автоматически показывает векторную иконку-заглушку
(`ImplantGlyph`) — как только файлы появятся, они подхватятся сами.
